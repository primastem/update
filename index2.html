Update PrimaSTEM firmware

1. Using Chrome or Edge, plug in your device to a USB port (make sure your USB-C cable supports data transfer).

2. Select your device and language:

  • Robot
  • Control board
  • Development / test

3. Hit button (Connect) BELOW and select the correct USB-COM port. (This step requires Chrome or Edge browser.)

This is a firmware update web page for PrimaSTEM devices.
